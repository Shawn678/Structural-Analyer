## 二、問題一：MDOF 勁度矩陣模型錯誤

### 現況

`seismic_logic.calculate_mdof_periods()` 組裝 K 矩陣的方式為**三對角矩陣（剪力框架模型）**：

```
K = [ k1+k2  -k2   0  ]
    [ -k2   k2+k3 -k3  ]
    [  0    -k3   k3   ]
```

### 問題

剪力框架模型假設各層之間有**水平樓板剛性連結，上層的位移是相對位移**。這適用於多層建築，但**不適用於多跨連續梁橋**。

橋梁的正確力學模型是**並聯懸臂（parallel cantilever）**：各橋墩底部固定，頂部透過剛性橋面連結（或各自獨立振動），每根墩的回復力只取決於自身的頂端絕對位移，不是相對位移。

#### 兩種模型比較

| 項目 | 剪力框架（現況，錯誤） | 並聯懸臂（正確） |
|---|---|---|
| K 矩陣形式 | 三對角 | 對角 |
| 位移定義 | 層間相對位移 | 墩頂絕對位移 |
| 適用場合 | 多層建築 | 多墩橋梁（剛性橋面） |
| 自然週期 | 偏短（勁度被高估） | 正確 |

#### 力學推導

設第 i 墩的側向勁度為 $k_i$（從 K_red 對角線取得，已含橋面耦合效應），墩頂位移為 $u_i$。

由於橋面剛性（rigid deck 假設），所有墩頂位移相同：$u_1 = u_2 = \cdots = u_n = u$

各墩回復力：$f_i = k_i \cdot u_i = k_i \cdot u$

整體回復力：$F = \sum k_i \cdot u = \left(\sum k_i\right) u$

因此等效單自由度系統：$K_{eff} = \sum k_i$，不是三對角矩陣。

**若橋面非剛性**（撓性橋面），墩頂位移不同，則 K 矩陣需從 FEM 完整縮減，不能用簡化的三對角或對角形式。

### 待實作

在 `seismic_logic.calculate_mdof_periods()` 加入 `model_type` 參數：

```python
def calculate_mdof_periods(masses, stiffnesses, model_type="rigid_deck"):
    # model_type = "rigid_deck"  → 對角 K（並聯懸臂）
    # model_type = "shear_frame" → 三對角 K（現況，建築用）
```

`rigid_deck` 模型下，廣義特徵值問題退化為單自由度：

$$T_1 = 2\pi\sqrt{\frac{\sum m_i}{\sum k_i}}$$

若需要多模態（撓性橋面），則需從 FEM 的完整 K_red 做 Guyan 縮減，不在此文件範圍內。

## 三、問題二：規範 2.4 節地震力分布

### 現況

目前計算 $V_L$、$V_T$ 後，沒有把力施加回結構，Tab C 顯示的是總力純量，無法得到構材內力。

### 規範要求

規範 2.4 節（等值靜力法）：

$$p_e(x) = k_h \cdot w(x)$$

其中：
- $p_e(x)$：沿橋軸的水平慣性力分布（kN/m）
- $w(x)$：沿橋軸的重量分布（kN/m）= 自重線載重
- $k_h = V / \int w(x)dx = V / W$：水平地震力係數（= $C_d$）

### 物理意義

地震慣性力正比於質量分布，重的地方受到更大的水平力。對連續梁橋，橋面本身有分布質量，應在**每根元素上施加均布水平力** $p_e = k_h \cdot (密度 \times A \times g)$（N/m），而不是集中在墩頂。

### 待實作流程

```
1. 計算 k_h = V / W（縱向與橫向各一個 k_h）
2. 對每根橋面元素：p_e = k_h × density × A × Le（均布力，N/m）
3. 對每根橋墩元素：p_e = k_h × density × A × Le（同上）
4. 組裝水平均布力向量，加入 truss_data["element_loads"]（橫向 w）
5. 重新執行 FEM 求解（run_numerical_analysis）
6. 從結果提取各元素內力
```

注意：w 方向在 element_loads 目前只有 `w`（local y2 方向），水平地震力需要確認元素局部座標的對應方向，或改用節點等效力施加。

## 四、問題三：規範 2.7 節構材內力三向組合

### 現況

目前在總力層級做：

```
組合一：1.0·V_L + 0.3·V_T + 0.3·V_v
組合二：0.3·V_L + 1.0·V_T + 0.3·V_v
組合三：0.3·V_L + 0.3·V_T + 1.0·V_v
```

這把三個方向的純量直接加總，**沒有物理意義**（三個力在不同方向，不能直接相加）。

### 規範要求

規範 2.7 節要求對**每根構材的每個內力分量**做組合。設縱向地震產生的某構材彎矩為 $M_L$，橫向為 $M_T$，垂直為 $M_v$，則：

```
組合一：1.0·M_L + 0.3·M_T + 0.3·M_v
組合二：0.3·M_L + 1.0·M_T + 0.3·M_v
組合三：0.3·M_L + 0.3·M_T + 1.0·M_v
```

軸力 N、剪力 V2/V3 分別同樣做三向組合，各自取控制值。

### 待實作流程

```
1. 縱向地震 FEM 求解 → element_forces_L（每根元素 N/V2/V3/M2/M3）
2. 橫向地震 FEM 求解 → element_forces_T
3. 垂直地震 FEM 求解 → element_forces_v
4. 對每根元素、每個內力分量，計算三組組合值，取最大
5. 輸出：每根元素的設計內力（控制組合）
```

垂直地震（$V_v$）的施加方式：垂直慣性力 $p_v = k_v \cdot w(x)$，方向為全域 Z。

### 與問題二的關係

問題二（FEM 重分析）是問題三的前置條件。必須先能把三個方向的地震力各自施回結構跑 FEM，才能在構材內力層級做組合。
