# 設計規格：橋梁箱涵斷面（Box Girder Section）

**日期**：2026-06-20  
**狀態**：草稿

---

## 1. 範疇

在現有材料／斷面系統中新增「箱涵」斷面形狀，支援單箱多室、直腹板、頂板懸臂，並於前端提供 Plotly 即時斷面預覽（含外廓尺寸標注）。

不在本次範疇：傾斜腹板、底板懸臂、預力鋼鍵換算斷面、時間相依行為。

---

## 2. 幾何定義

```
←──────────────── b_top ────────────────→
←─ c_top ─┬──────────────────────┬─ c_top ─→
          │  ██████████████████  │  ← t_top
          │  ┌──┐    ┌──┐    ┌──┐  │
          │  │  │    │  │    │  │  │  hw = h - t_top - t_bot
          │  │  │    │  │    │  │  │
          │  └──┘    └──┘    └──┘  │
          │  ████████████████████  │  ← t_bot
          └────────────────────────┘
←──────────────── b_bot ────────────────→

（範例為 n_cell = 3）
外腹板厚：t_web；內隔板厚：t_dia
```

### 幾何參數

| 參數 | 說明 | 單位 |
|---|---|---|
| `b_top` | 頂板總寬（含懸臂） | m |
| `b_bot` | 底板總寬 | m |
| `h` | 箱涵總高 | m |
| `t_top` | 頂板厚 | m |
| `t_bot` | 底板厚 | m |
| `t_web` | 外側腹板厚 | m |
| `t_dia` | 內隔板厚 | m |
| `n_cell` | 室數（1～5） | — |
| `c_top` | 頂板單側懸臂長 | m |

衍生幾何：
- 箱涵淨寬（兩腹板內緣）：`b_box = b_bot - 2*t_web`（底板側定義）
- 腹板高（內淨高）：`hw = h - t_top - t_bot`
- 隔板間距（均分）：`s = b_box / n_cell`

## 3. 斷面性質計算（`core/materials.py`）

### 3.1 面積 A

各板件分開計算後加總：

```
A = b_top * t_top                          # 頂板（含懸臂）
  + b_bot * t_bot                          # 底板
  + 2 * t_web * hw                         # 外腹板（×2）
  + (n_cell - 1) * t_dia * hw             # 內隔板
```

### 3.2 慣性矩 I33（對水平形心軸）

各板件以形心為參考，套用平行軸定理。形心高度 `ȳ` 由加權平均得出。

每個板件貢獻：`I_self + A_piece * d²`，其中 `d` 為板件形心距全斷面形心之距離。

### 3.3 慣性矩 I22（對垂直形心軸）

同理，對垂直軸做平行軸定理。注意頂板懸臂在 I22 貢獻較大。

### 3.4 扭轉常數 J（Bredt 多室薄壁）

對 `n_cell` 室建立聯立方程組，每室 `i` 的相容方程：

```
Σ_j [ δᵢⱼ * qⱼ ] = 2 * Aᵢ
```

其中：
- `Aᵢ`：第 `i` 室封閉面積
- `δᵢᵢ = Σ(ds/t)` 沿第 `i` 室周長（包含相鄰隔板各取一半）
- `δᵢⱼ = -共用隔板長/t_dia`（相鄰室之耦合項）

以 `numpy.linalg.solve` 求解剪流向量 `q`，再得：

```
J = 2 * Σᵢ ( qᵢ * Aᵢ ) / (單位扭矩)
```

實作上統一取單位扭角率（Batho–Bredt 完整推導）輸出 `J`（長度⁴）。

## 4. 前端 UI（`app.py`）

### 4.1 斷面參數輸入

當 shape 選擇 `"箱涵"` 時，顯示以下輸入欄位：

- `n_cell`：`st.selectbox`，選項 1～5
- 其餘 8 個幾何參數：`st.number_input`，預設值參考常見橋梁尺寸

欄位排列採 `st.columns(2)` 兩欄配置，與現有其他斷面 UI 風格一致。

### 4.2 即時斷面預覽（Plotly）

位置：參數輸入欄位右側（`st.columns([1, 1])`）。

內容：
- 用 `go.Scatter`（filled polygon）繪製斷面外廓與各室空腔
- 外廓：灰色填充；空腔：白色填充（疊加遮蓋）
- 尺寸標注（`go.layout.Annotation` + 箭頭）：`b_top`、`b_bot`、`h`、`c_top`
- 圖形等比例（`yaxis.scaleanchor="x"`），無座標軸刻度

觸發：Streamlit 參數改變即自動 rerun，圖形即時更新。

## 5. 與現有系統整合

- `compute_section_props`：新增 `if shape == "箱涵":` 分支，回傳相同的 `{"A", "I33", "I22", "J"}` 字典，介面與現有形狀完全一致。
- `expand_truss_data`、`compute_self_weight`：**不需修改**，箱涵只是多一種 shape。
- 現有 Custom / 矩形 / 圓形 / I 形等斷面：**完全不受影響**。

---

## 6. 測試重點

- `n_cell = 1`：退化為單室箱形，J 應與 Bredt 單室公式吻合
- `n_cell = 2, 3`：與手算或文獻值比對
- `c_top = 0`：無懸臂，I22 應對稱
- 極端薄壁（t 趨近 0）：確認不除以零
